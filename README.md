### **Compare and search string algorithms for JavaScript**
"Алгоритмы сравнения и поиска строк."

## Usage
```js
var strike_match = require('../nice_alg').strike_match;

function example() {
 // нечеткое сравнение строк
 console.log(strike_match("Sealed","Healed") === 0.8);
}

example();

```
